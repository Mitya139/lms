 # apps/templates — shared Django/Jinja2 templates

 Purpose
 - Centralized directory for UI templates used by apps (admin, forms, emails, learning, users, courses, etc.).

 Subtrees
 - admin/ — admin UI customization (e.g., base_site.html)
 - forms/ — shared form/dialog templates.
 - emails/ — emails (activation, notifications, alumni_promotion, etc.).
 - learning/** — learning pages (enrollment, invitation, teaching/*, event_detail, etc.).
 - users/** — user pages/forms (application, widgets, etc.).
 - courses/** — course UI fragments.
 - crispy_forms/, bootstrap_pagination/ — third‑party/custom include templates.
 - international_schools/** — landing pages for international schools.
